# Health Tracking App

A MERN stack application for tracking health records.

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/HealthTrackingApp.git
cd HealthTrackingApp

Install Dependencies

#Backend:
cd backend
npm install

#Frontend:
cd ../frontend
npm install


Start the Application

cd ../backend
npm start

cd ../frontend
npm run dev
